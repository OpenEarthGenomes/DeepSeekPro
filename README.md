# 🚀 DeepSeek Pro

**Ingyenes, nyílt forrású Android alkalmazás a DeepSeek AI-hoz.**

## ✨ Funkciók

- 💬 **Chat** – WebView alapú DeepSeek integráció
- 🧠 **DeepThink (R1)** – Kapcsolható a lassabb, komplexebb modell
- 📁 **Jegyzettömb** – Minden beszélgetés automatikusan mentésre kerül
- 🔍 **Keresés** – Két szintű keresés a beszélgetésekben
- 📎 **Fájlfeltöltés** – Dokumentumok, képek csatolása
- 🌐 **Magyar nyelv** – Automatikus nyelvi kényszerítés
- ⚡ **Gyors lapozás** – Csúszka a jobb oldalon
- 📤 **Megosztás** – Beszélgetések exportálása
